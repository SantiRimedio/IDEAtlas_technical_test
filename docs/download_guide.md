# Data Download Guide

## Prerequisites

1. Google Earth Engine account: [signup.earthengine.google.com](https://signup.earthengine.google.com)
2. Authenticate GEE:
   ```python
   import ee
   ee.Authenticate()
   ee.Initialize(project='your-project-id')
   ```

## Downloading Data

### Sentinel-2 Composites

Use `notebooks/01_download_satellite.ipynb` or directly:

```python
from src.downloads.download_s2 import quick_download_s2

quick_download_s2(
    aoi_bounds=(minx, miny, maxx, maxy),  # WGS84
    start_date='2020-06-01',
    end_date='2020-08-31',
    output_filename='datasets/raw/s2_composite.tif',
    bands=['B2', 'B3', 'B4', 'B8', 'B11', 'B12'],
    cloud_filter=20,
    apply_harmonization=True,
    normalize=True
)
```

For large areas (>40MB), the downloader auto-tiles and mosaics. For very large areas, use `export_to_drive()` instead.

### GHSL Built-Up Data

Use the `GHSLDownloader` class in `src/downloads/download_ghsl.py`. Downloads from the `JRC/GHSL/P2023A/GHS_BUILT_S` collection.

### Reference Labels

Created from RENABAP informal settlement polygons (`data/boundaries/renabap-datos-barrios.geojson`) rasterized to match S2 resolution. See `create_reference_labels_quick()` in `src/downloads/download_ghsl.py`.

### Building Footprints

Downloaded from Google Open Buildings v3 via `src/downloads/building_downloader.py` or from Microsoft Planetary Computer.

### Morphometric Features

Run `notebooks/02_download_morphometrics.ipynb` to compute building density at multiple scales (10m, 50m, 100m) using `src/downloads/morphometric_calculator.py`.

## File Placement

After downloading, place files in:

```
datasets/
├── raw/
│   ├── s2_2020_AMBA.tif              # S2 composite (year 1)
│   ├── s2_2025_AMBA.tif              # S2 composite (year 2)
│   ├── s2_2025_building_density.tif  # Density raster
│   ├── reference_labels_*.tif        # Ground truth
│   ├── ghsl_builtup_*.tif            # GHSL data
│   └── density_multiscale_AMBA/      # Multi-scale density stack
├── patches/                          # Generated by extract_patches
│   ├── train/
│   ├── val/
│   └── test/
└── predictions/                      # Generated by inference
```

## Validation

After downloading, validate compatibility:
```python
from src.downloads.data_validator import quick_validate_dataset
quick_validate_dataset('datasets/raw/')
```

Or check a single file:
```python
from src.downloads.download_s2 import validate_s2_compatibility
validate_s2_compatibility('datasets/raw/s2_composite.tif')
```
